# GTM.Garden Market — Skill Library

Shareable Claude Code skills for common business owner use cases. Built by GTM.Garden.

## How to Use

Clone this repo into your Claude Code project, then run the skill you need.

## Available Skills

| Skill | Command | What it does |
|---|---|---|
| SFDC Setup | `/sfdc-setup` | Connects Claude to your Salesforce org via Agentforce DX |
| SFDC Reports | `/sfdc-reports` | Build reports and dashboards by describing them in plain English |
| SFDC Migrate | `/sfdc-migrate` | Migrate leads, contacts, accounts, and opps from any CRM into SFDC |

## Client-Specific Versions

Pre-configured skills tailored to a specific client's stack live in `clients/[client-name]/`.

These have the client's tools, org names, and context pre-loaded — drop them in and run.
